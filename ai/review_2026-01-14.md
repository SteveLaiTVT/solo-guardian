## 数据模型设计

### CheckIn 模型
```
- id: UUID
- userId: 外键关联 User
- checkInDate: 日期字符串 (YYYY-MM-DD)
- checkedInAt: 签到时间戳
- note: 可选备注 (最多500字)
- 唯一约束: userId + checkInDate (每天每人只能签到一次)
```

### CheckInSettings 模型
```
- id: UUID
- userId: 外键关联 User (唯一)
- deadlineTime: 截止时间 (默认 "10:00")
- reminderTime: 提醒时间 (默认 "09:00")
- reminderEnabled: 是否启用提醒 (默认 true)
- timezone: 时区 (默认 "Asia/Shanghai")
```

## API 端点

| 方法 | 路径 | 描述 |
|------|------|------|

| GET | /api/v1/check-ins/today | 获取今日签到状态 |
| GET | /api/v1/check-in-settings | 获取签到设置 |
| PUT | /api/v1/check-in-settings | 更新签到设置 |
